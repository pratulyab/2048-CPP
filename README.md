# 2048 Game in C++

- The famous 2048 Game coded in C++, played on console.

- Controls: 

	a / A    - Left
	s / S    - Down
	w / W    - Up
	d / D    - Right
	u / U    - Undo (Reverts the grid to the previous state)
	n / N    - New Game
	Q        - Quit
